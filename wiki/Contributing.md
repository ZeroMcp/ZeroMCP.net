# Contributing

Contributions are welcome.

---

## How to contribute

- Open issues for bugs or feature ideas.
- Submit pull requests with clear descriptions and, where relevant, tests.
- Keep the main **README.md** and **wiki** in sync when adding features or options; update **progress.md** for notable changes (per project conventions).

---

## High-impact next steps

Ideas that would add the most value:

1. **SSE transport support** — Allow MCP over Server-Sent Events in addition to Streamable HTTP.
2. **Richer minimal API parameter binding** — Query and body binding from the route delegate so minimal API tools match controller flexibility.
