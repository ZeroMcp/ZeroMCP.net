## Using this wiki

- **In-repo:** Open any `.md` file; links use wiki page names without `.md` (e.g. `[Quick Start](Quick-Start)`). Some renderers (e.g. GitLab wiki) resolve these when the page is named `Quick-Start.md` or `Quick-Start`.
- **GitLab Wiki:** Create a new wiki in your GitLab project and copy the contents of each file into a page with the same title (e.g. "Quick-Start" or "Quick Start"). Adjust links if your wiki uses different URL slugs.
- **GitHub Wiki:** Same idea — create a wiki repo and add pages; link between them by page name.
